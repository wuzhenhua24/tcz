## 第 1 章 Scope（范围）概要

### 文档适用范围

本规范定义了 5G 系统（5GS）中使用的非接入层（NAS）过程，主要涵盖：

1. **5GMM（5GS 移动性管理）协议**
   - UE 与 AMF（接入和移动性管理功能）之间的移动性管理
   - 支持 3GPP 接入和非 3GPP 接入
   - 为 NAS 协议提供安全控制

2. **5GSM（5GS 会话管理）协议**
   - UE 与 SMF（会话管理功能）之间的会话管理
   - 支持 3GPP 接入和非 3GPP 接入
   - 处理 5GS PDU 会话
   - 与接入层的承载控制一起控制用户面资源

3. **系统间移动性支持**
   - NG-RAN 与 E-UTRAN 之间
   - NG-RAN 与连接到 EPC 的非 3GPP 接入网络之间
   - 连接到 5GCN 的非 3GPP 接入网络与 E-UTRAN 之间
   - NG-RAN 与连接到 5GCN 的非 3GPP 接入网络之间

4. **UE 策略传递服务**
   - UE 与 PCF（策略控制功能）之间
   - 支持 3GPP 接入和非 3GPP 接入

### 适用实体

本文档适用于 5GS 中的以下网元：
- UE（用户设备）
- AMF（接入和移动性管理功能）
- SMF（会话管理功能）
- PCF（策略控制功能）

## 总结

本章节是 3GPP TS 24.501 规范的参考文献列表，涵盖了 5G 系统 NAS 协议所需的所有相关标准和规范。参考文献来源广泛，包括：

- **3GPP 规范**：占主体，涵盖从服务需求到协议实现的各个层面
- **IETF RFC**：提供互联网协议基础（TCP/UDP、IP、安全、头部压缩等）
- **IEEE 标准**：提供底层网络和时间同步标准
- **ITU-T 和 BBF**：提供识别计划和管理协议

这些参考文献共同构成了 5G 系统 NAS 协议的完整技术基础，体现了 5G 系统的复杂性和与现有标准的继承关系。

## 关键特性

1. **版本管理清晰**：明确区分具体引用和非具体引用，确保标准一致性
2. **覆盖全面**：从底层传输到应用层服务，从安全到移动性管理，各方面都有详细的参考标准
3. **标准演进**：既包含 4G（E-UTRAN）相关标准，也包含 5G（NR）新标准，支持系统间互操作
4. **特色功能**：V2X、UAS、ProSe、TSN 等新兴应用都有专门的标准支持
